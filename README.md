# Bid Zone

Bid Zone is a terminal-based auction system inspired by the IPL auction model.
It simulates a real-time bidding environment where teams compete to buy players
within a fixed purse.

# Core Idea

- Players are auctioned one by one
- Teams place bids until a playerâ€™s auction is finalized
- Players are marked as sold or unsold
- Every auction starts from scratch (no previous history)

# Features

- Team-based bidding system
- Authentication using team credentials to participate in bidding
- Player-wise bidding with auction finalization
- Fully terminal-based interaction
- IPL-style bidding rules
  - Range-wise price increment
  - No random or arbitrary bids allowed

# Team Functionalities

- Remaining Purse
- Bid on players
- My Purchases
- View bid history
